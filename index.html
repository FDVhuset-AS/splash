Nanosplash
Small. Simple. Reliable.
ns.show('text').inside('#myDiv')
Give user progress feedback
ns.img(src).progress(
  [promise1, 'Loading A'],
  [promise2, 'Loading B'],
  [promise3, 'Loading C'],
).inside('#myDiv')
Handle async functions
ns.while(promise).show('Loading ...').inside('#myDiv')
Display fullscreen
const splash = ns.show('Fullscreen text')
splash.moveTo('#myDiv')
splash.remove()
Display multiple, independent instances
ns.img(src).show('Loading component A').inside('#a')
ns.show('Loading component A').inside('#b')
ns.show('Loading component A').inside('#c')


ns.hide() // Removes instances in a FIFO manner.
ns.hideAll() // Removes all instances.
