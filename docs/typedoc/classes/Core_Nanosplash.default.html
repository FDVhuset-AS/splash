hide

  • hide(ref?: Destination): void

  • Implementation of NanosplashInterface.hide

    • Defined in Core/Nanosplash.ts:197

    Hide

    Delete specific Splash instance or the latest one. There are three different ways of using this function.

    Use SplashInstance IDs

    Every new splash instance is given a unique, randomized ID. This ID is easily retrieved by using the getId method.
    const splashInstance = ns.show('Loading');
    ns.hide(splashInstance.getId());

    Use CSS Selector

    Pass the selector referring to the element in which the splash instance resides.
    ns.show('Loading').inside('#my-div');
    ns.hide('#my-div');

    Pass no argument

    When calling the hide method without passing arguments, it will delete the latest splash instance added to the DOM, in other words, in a LIFO fashion.
    ns.show('Loading');
    ns.hide();

    Parameters

    • Optional ref: Destination

      This can either be a SplashInstance ID or a CSS selector referring to the element in which the instance is residing.

    Returns void
