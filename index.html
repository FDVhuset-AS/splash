API Reference Documentation
Give user progress feedback
ns.progress(
  [promise1, 'Loading A'],
  [promise2, 'Loading B'],
  [promise3, 'Loading C'],
).inside('#myDiv')
Handle async functions
ns.while(promise).show('Loading ...').inside('#myDiv')
Display fullscreen
const splash = ns.show('Fullscreen text')
splash.moveTo('#myDiv')
splash.remove()
Display multiple, independent instances
ns.show('Loading component A').inside('#a')
ns.show('Loading component B').inside('#b')
ns.show('Loading component C').inside('#c')


ns.hide() // Removes instances in a LIFO manner.
ns.hideAll() // Removes all instances.
